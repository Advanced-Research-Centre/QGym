QGym

A gymnasium environment for quantum circuit optimization

Installation

  1. Clone the repository
git clone https://github.com/Advanced-Research-Centre/QGym
  1. Install the QGym environment package. In the main project directory type:
pip3 install -e .
  1. (Optional) Install the remaining packages, if you do not have them already
pip3 install qiskit gymnasium jupyter
  1. Get acquianted with the Usage.ipynb file.
jupyter notebook
